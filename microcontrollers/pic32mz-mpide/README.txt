Example of generic Makefile to build MPIDE sketches from command line.

 1) Install MPIDE from:
    http://chipkit.net/started/install-chipkit-software/

 2) Setup APP value in Makefile to name of your sketch.

 3) Check that MPIDE value in Makefile matches your MPIDE installation.

 4) To build the application, run:

        make

 5) To upload the application to the WiFire board, run:

        make load
